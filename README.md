# 카드 뒤집기 게임

React + TypeScript + Vite로 만든 동물 카드 매칭 게임입니다.

## 🚀 기능

- 12지 동물 카드 매칭 게임
- 카드 섞기 및 게임 리셋

## 🛠️ 기술 스택

- **Frontend**: React 19 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Testing**: Vitest


**배포 URL**: https://bangdler.github.io/card-flip/

## 🎮 게임 규칙

1. 같은 동물 카드를 찾아 매치하세요
3. 매치할 때마다 10점을 얻습니다
4. 모든 카드를 찾으면 게임 클리어!
